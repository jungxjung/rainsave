Rain SAVE 儲集框架 為一種塑膠射出成型製品其廣泛運用於地下儲集雨水及滲透保水等基地保水之工程使用，我司所生產之儲集框架產品具有精度高、易於安裝、抗壓強度高、質量輕盈等特性。產品採100%均質一級回收塑料並添加各式耐衝擊、抗紫外線、抗凍融等助劑製成。

Rain SAVE 儲集框架三大優勢：
1.產品成分比例可依據客戶不同需求批量生產。
2.產品顏色可客製化。
3.本產品為一體成型射出式，安裝容易，無需使用配件即可施工。

產品施作導覽

RAIN SAVE為一種應用於地表面下的框架式雨水收集系統，其儲水率達95%以上，材質為可回收聚丙烯，此系統主要由儲集框架主體與儲集框架側板所建構而成。
此框架系統的儲水量體大小、型體形狀與深度可依施工現場做調整，為一種開放式的組合系統。
本塑膠儲集框架建材廣泛應用於地下儲水工程項目，經由組合後每m2由頂部延伸至底部具備20個柱狀式結構，最大可承載25噸汽車通行[註1] 。
此產品可提供基地保水Q5的地下貯集滲透設施計算使用。

知識分享

如何在儲集框架、混凝土水箱、PP(FRP)桶槽型水箱中做選擇?

施作10立方~1000立方的滯洪儲水、滲透水設施建議使用正在旺所生產的RAIN SAVE 儲集框架產品，具備施工快速、無須配件，高永續性、均質強度、槽體不易滋生沼氣等優點

為何使用 Rain SAVE 儲集框架? (抗壓強度探討)

在結構上來說，各家產品的結構設計對於應付T-25承載基本上並無太大問題；而在添加劑的配比上，經過各項研究顯示添加10%~20%的GF時應用於工程上是非常有幫助的

為何使用 Rain SAVE 儲集框架? (進水速度與地震帶上的適應性探討)

若考量雨水流速(進水速度)，長短板型式及框架型式因內部結構空間較大，是較理想的選擇；而若是考慮到產品於亞洲地震帶上的適應性，框架型式整體的結構複雜程度、產品設計都較為完善，比較能適應地震時產生之各種力量

Rain SAVE 儲集框架埋設於地下能否抗重車壓?

根據 Rain Save 工法與長期性能，垂直方向荷重適用範圍是以土的單位體積重量18kN/m3與活荷重(指車輛荷重以及集聚荷重T-25)進行評定；水平方向荷重適用範圍是以土的單位體積重量18kN/m3與地表承載荷重10kN/m2進行評定

Rain SAVE 儲水型儲集框架的不透水布如何鋪設? 整體周圍的不織布如何包覆?

內有不透水布燒黏、鋪設以及不織布搭接鋪設之詳細影片說明

Rain SAVE 儲集框架該如何安裝? 能否獨立安裝?

Rain SAVE 儲集框架具備多項優良設計特點和防呆設計，只需一層一層上下堆疊以及左右水平靠接即可輕鬆完成主體安裝，側板自帶卡勾也能輕易完成側面封板。只要謹遵使用須知頁面內的說明，人人都能輕鬆自行完成安裝。

Rain SAVE 儲集框架安裝時有哪些該注意的事項?

鋪設基礎需進行嚴格整平作業，尤其是有人工地盤的地方；凸起物需打除，凹洞處需以相同材質填實。若沒有確實遵照須知事項內的說明施作，可能造成不理想的結果(內有案例說明)。

如何知道根據自己的需求需要使用多少數量的Rain SAVE 儲集框架材料?

每施作一立方米會需要16個儲集框架主體；施作側面封板每一平方米則需要4片儲集框架側板。若您施作形式為儲水型設施則會多出一項不透水布材料，而其餘工程材料以及詳細的單價分析計算可至需求計算頁面查詢。

Rain SAVE 儲集框架是否容易淤積? 若淤積該如何清理?
